# LiveNote アプリケーション

## 概要
このプロジェクトは、Spring Bootを使用して開発されたアーティスト情報管理のバックエンドAPIです。
フロントエンド（HTML/CSS/JavaScript）と連携し、データの追加、表示、削除が可能です。

## 使用技術
- **バックエンド**: Java 11, Spring Boot, Spring Security
- **データベース**: MySQL
- **ドキュメント**: Springdoc OpenAPI

## 機能一覧
- 音楽ニュース（お気に入り登録）
- ユーザーの認証・認可機能（一般ユーザー）

## APIドキュメント
アプリケーションを起動後、以下のURLにアクセスしてAPI仕様をご確認いただけます。
`http://localhost:8080`
